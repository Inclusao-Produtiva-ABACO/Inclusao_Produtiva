# 🚀 Inclusão Produtiva - ABACO

Este projeto é desenvolvido como parte da **Atividade de Extensão** do 7º período do curso de **Engenharia de Computação**. O objetivo é fornecer uma solução tecnológica para a **ABACO (Associação Beneficente de Auxílio à Comunidade)**, otimizando a gestão de seus cursos de capacitação.

---

## 📌 Objetivo do Projeto
Desenvolver uma plataforma integrada para facilitar a inclusão produtiva, automatizando processos manuais e permitindo um controle eficiente de alunos, frequências e recursos materiais da associação.

## 🛠️ Funcionalidades Principais

* 📋 **Inscrição de Membros:** Portal dedicado para que alunos se candidatem aos cursos oferecidos.
* 👨‍🏫 **Diário de Classe Digital:** Sistema de gerenciamento de faltas e presenças para os instrutores.
* 📜 **Certificação Automática:** Geração de certificados em PDF baseada nos critérios de frequência.
* 📦 **Controle de Insumos:** Mini gerenciamento de estoque para compra e uso de materiais didáticos.

---

## 🏗️ Engenharia de Software e Tecnologias

O projeto segue princípios de engenharia de software para garantir escalabilidade e fácil manutenção. Estamos avaliando as seguintes stacks:

* **Frontend:** React com Tailwind CSS / HTML5 + Tailwind
* **Backend:** Python (FastAPI/Django) ou Node.js
* **Banco de Dados:** PostgreSQL (via Supabase) ou SQLite.
* **Controle de Versão:** Git & GitHub (GitFlow)

> **Nota Técnica:** Optamos por migrar da ideia inicial de planilhas para um banco de dados relacional para garantir a integridade referencial e consistência dos dados, especialmente na geração de certificados e controle de estoque.
