# Inclusao_Produtiva
O objetivo é construir uma plataforma de inclusão produtiva para a ABACO, facilitando a gestão de cursos de capacitação, controle de frequência e manejo de insumos. Este projeto faz parte da Atividade de Extensão do 7º período de Engenharia de Computação do IFSULDEMINAS, campus Poços de Caldas.
