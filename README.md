Nome do Projeto: Inclusão Produtiva

Objetivo: Desenvolver uma plataforma de inclusão produtiva para a ABACO, facilitando a gestão de cursos de capacitação, controle de frequência e manejo de insumos. Este projeto faz parte da Atividade de Extensão do 7º período de Engenharia de Computação.

Principais Funcionalidades:

📋 Inscrição de Membros: Portal para alunos se candidatarem aos cursos.

👨‍🏫 Diário de Classe Digital: Gestão de faltas e presenças simplificada para professores.

📜 Certificação Automática: Geração de certificados baseada na frequência mínima.

📦 Controle de Insumos: Gerenciamento de estoque para materiais didáticos e operacionais.
